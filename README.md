List
I learned that HTML provides three types of lists to organize content clearly:
Ordered lists: items are numbered and used when order is important (e.g., steps in a process).
Unordered lists: items use bullet points and are used when order is not important.
Definition lists: include terms and their corresponding definitions.
These list types help present information in a structured and readable way.
Ordered Lists
<ol> : The ordered list is created with the element.
<li> : Is used to create each item in the list, and the browser automatically indents the list by default.
Unordered Lists
<ul> : The unordered list is created with the element.
<li> : Is used to create each item in the list, and the browser automatically indents the list by default.
Definition Lists
<dl> : Used to create list definitions
<dt> : Contains the term that needs to be defined
<dd> : Contains the definition

Nested Lists
You can place a list inside a <li> to create a nested list.
The browser will indent it deeper than the parent list. With nested <ul> lists, the dots usually restyle themselves.


Summary Lists
There are three types of HTML lists: ordered, unordered, and definition. 
Ordered lists use numbers.
Unordered lists use bullets.
Definition lists are used to define terminology.
Lists can be nested inside one another.
